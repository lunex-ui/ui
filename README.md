# Lunex UI


[![Version](https://img.shields.io/badge/version-0.1.0-black)](https://github.com/lunex-ui/ui/releases)
[![License](https://img.shields.io/badge/license-Apache%202.0-black)](./LICENSE)
[![Stars](https://img.shields.io/github/stars/lunex-ui/ui?style=social)](https://github.com/lunex-ui/ui/stargazers)

Lunex UI is an open-source component library for modern frontend applications. The project starts with React and Next.js, with a longer-term goal of expanding into a broader UI system that can grow across more frameworks over time.

The focus is simple: clean components, a flexible theme system, and a developer experience that feels fast without becoming messy.

## Getting Started

Clone the repository and install dependencies:

```bash
pnpm install
```

Start the Next.js playground:

```bash
pnpm dev:playground-nextjs
```

If you want the package watch process running alongside the playground:

```bash
pnpm dev
```

## Contributing

Contributions are welcome.

If you want to contribute:
- open an issue for bugs, ideas, or API discussions
- keep pull requests focused and easy to review
- match the existing code style and naming patterns
- test changes in the playground before opening a PR

Good first contribution areas:
- new foundational components
- accessibility improvements
- better theme tooling
- docs and examples
- bug fixes and DX improvements

See [CONTRIBUTING.md](./CONTRIBUTING.md) for branch strategy, PR expectations, and the recommended contribution flow.
